# E-Library System

## Project Description
The E-Library System is a full stack web application that allows users to manage a collection of books. It features a React-based frontend and a Node.js/Express backend, providing a seamless experience for adding, viewing, and managing books in a digital library.

## Setup Instructions

### Prerequisites
- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

### 1. Clone the Repository
```
git clone <repository-url>
cd E-Library--System
```

### 2. Install Dependencies
#### Backend
```
cd server/backend
npm install
```
#### Frontend
```
cd ../../client/frontend
npm install
```

## How to Run the Application

### 1. Start the Backend Server
```
cd server/backend
npm start
```
By default, the backend runs on [http://localhost:5000](http://localhost:5000).

### 2. Start the Frontend Application
Open a new terminal window/tab and run:
```
cd client/frontend
npm start
```
The frontend will be available at [http://localhost:3000](http://localhost:3000).


For more details, refer to the `README.md` files in the `client/frontend` and `server/backend` directories.
